To use this application, follow these steps:

1. Install the required libraries by running the following command in your terminal: pip install -r requirements.txt
2. Run the code to your local machine.
3. Open the command prompt or terminal and navigate to the directory containing the code.
4. Run the following command to start the application: streamlit run app.py
5. Upload the depthmap image of the child in JPG, PNG, or JPEG format using the file uploader.
6. Click on the "Predict" button to generate the prediction.
